> A fast, responsive movie & TV show explorer featuring a universal details page, official YouTube trailer modal, and cinematic design — built with clean, modern React.

## Features
- **Universal details page** — works for both movies and TV series  
- **Smart auto-detection** — automatically detects media type from TMDB ID  
- **Official YouTube trailer** — displayed in a sleek full-screen modal  
- **Fully responsive design** — looks perfect on mobile, tablet, and desktop  
- **Modern React architecture** — clean, performant, and maintainable codebase  
- **Robust UX states** — loading indicators, error handling, and graceful fallbacks  
- **Horizontal scrollable rows** — Netflix‑style navigation with left/right arrows  
- **Watchlist (My List)** — save and manage your favorite movies & shows  
- **Integrated search** — directly accessible from the NavBar  

## Tech Stack
- **React 18 + Vite** — blazing fast development environment  
- **React Router v6** — modern client-side routing  
- **Redux Toolkit** — state management for search, favorites, and loading states  
- **Axios** — lightweight, flexible HTTP client with interceptors  
- **Tailwind CSS** — utility-first styling for rapid & consistent UI  
- **TMDB API** — rich movie & TV data source  
- **Heroicons** — modern SVG icons for navigation arrows  
- **react-icons/fa** — elegant Font Awesome icons (solid style)  
- **Vercel** — fast, zero‑config deployment
